# DeftBrain USPTO Trademark Application — Filing Prep

**Target filing**: Standard character word mark for `DEFTBRAIN` in Class 42, under Section 1(a) "use in commerce."
**Target cost**: $350 (base fee, no surcharges).
**Where you file**: https://teas.uspto.gov/ → "Initial application form"
**Filing as**: Individual, US citizen (LLC decision deferred; can assign later).

## Section 1 · The Mark

| Field | Value |
|---|---|
| Mark text | `DEFTBRAIN` |
| Mark type | Standard character mark |
| Description of mark | "The mark consists of standard characters, without claim to any particular font style, size, or color." |

**Why standard character, not the logo**: Protects the word in *any* font, color, or styling. Broader enforcement rights. If you ever redesign the visual identity, the trademark still applies. The logo can be filed separately later as a design mark if you want extra protection on the specific visual.

---

## Section 2 · Classification

**Class 42 only** — Scientific and technological services (SaaS, AI-powered software services).

Do **not** add Class 9 (downloadable software) or Class 35 (business services). You'd pay an extra $350 per class and have to prove use of each. Keep the filing lean and honest.

---

## Section 3 · Goods/Services Identification ⚠️ Critical

The USPTO charges a **$200 surcharge per class** if you use free-form (custom) language instead of pre-approved wording from the Trademark ID Manual. Stay inside the manual.

**ID Manual search**: https://idm-tmng.uspto.gov/

**Recommended starting identification** (assembled from ID Manual patterns — verify each exact phrase in the manual before submitting):

> Providing temporary use of non-downloadable software using artificial intelligence for generating written communications, decision support, and personal productivity; Software as a service (SAAS) services featuring software using artificial intelligence for generating written communications, decision support, and personal productivity.

**How to verify**: Open the ID Manual, search each phrase. Keep only phrases that appear verbatim (or use the manual's near-equivalent). If a phrase you want isn't there, either find the closest pre-approved alternative or accept the $200 surcharge for that phrase.

**Guardrails**:
- Stay under **1,000 characters total** (another $200 surcharge per extra 1,000 chars)
- Cover what you actually do, not aspirational future services
- Broad enough to cover communication, decision, planning, and analysis tools
- Narrow enough that every listed service actually exists on the site today

---

## Section 4 · Filing Basis

**Section 1(a) — Use in Commerce.** DeftBrain is live at deftbrain.com and accessible nationwide. Do not file under Section 1(b) "intent to use."

## Section 5 · Dates of First Use

Two dates required, typically the same for SaaS websites:

| Field | Value |
|---|---|
| Date of first use anywhere | `[YYYY-MM-DD in 2026 — see "How to pin down the date" below]` |
| Date of first use in commerce | `[Same date — web-accessible SaaS is in interstate commerce from launch]` |

**How to pin down the date**: Check these in order until one anchors the date within a week:
1. Railway deploy history — first production deploy of the site
2. First GA / analytics pageview from an IP outside Massachusetts
3. First social post, newsletter, or email announcing the launch
4. First real user generation log (a tool actually being used by someone)
5. Your first production commit to the `main` branch that deployed publicly

**What counts as "first use"**: The moment someone outside your household could load the site and use a tool. Soft launches to friends in other states count. Deploys to a staging URL don't. The domain registration date alone doesn't count — the site has to have been live and functional.

**Accuracy standard**: You're signing under penalty of perjury. "Approximately March 2026" is acceptable if you genuinely can't pin it closer. "March 15, 2026" is better. Don't pick a date you can't defend if challenged.

## Section 6 · Specimen of Use

A screenshot proving the mark is being used with the services.

**Recommended specimen**: Full-page screenshot of `deftbrain.com` homepage showing:
1. The BrandMark (logo + "DeftBrain" wordmark) clearly visible in the header
2. The tool grid/catalog below (proves services are being offered under the mark)
3. The browser URL bar showing `deftbrain.com` (authenticates source)

**Format**: PNG, submitted as an attachment in the TEAS form.

**Specimen description field**: "Screenshot of applicant's website at deftbrain.com showing the mark DEFTBRAIN as used in connection with the services."

**Avoid**:
- ❌ Isolated logo image (must show the mark connected to a service being offered)
- ❌ Mockup or design-tool export (must be the live site)
- ❌ A page that doesn't make clear what service is being provided

## Section 8 · Attorney

**Not required.** You're a US citizen filing for yourself, which qualifies for DIY self-representation.

If the examining attorney issues an office action that goes beyond clerical fixes (e.g., a substantive likelihood-of-confusion refusal or descriptiveness refusal), consider hiring a trademark attorney for that specific response — typically $300–600 flat fee. For the initial filing, a clean DIY application with ID Manual language will almost certainly go through without needing an attorney.

## Section 9 · Declaration & Signature

You sign under penalty of perjury attesting:
- You are the owner of the mark
- The mark is in actual use in commerce on the filing date
- To your knowledge, no other party has the right to use a confusingly similar mark
- All statements in the application are true

**Do not file** if any of these aren't clean. Fraud in a trademark application can result in the registration being cancelled later.

## Common Pitfalls to Avoid

1. **Overclaiming services.** Don't list services you don't actually provide. A competitor can cancel the registration for fraud.
2. **Free-form identifications.** The $200/class surcharge is real. Use the ID Manual.
3. **Weak specimen.** A logo in isolation isn't a specimen. Show the mark connected to the service being offered.
4. **Wrong entity.** Filing as yourself then assigning to an LLC works but adds paperwork and timing risk. Try to file in the ultimate owner's name.
5. **Ignoring office actions.** If an examining attorney issues one, you have 3 months to respond. Missing it abandons the application — you lose the $350 and have to start over.

## Optional Follow-Up: Design Mark (The Logo)

Once the word mark issues cleanly, consider filing a second application as a "special form" (design) mark to protect the specific visual: the brain-glasses illustration + gold "D" + navy wordmark. Another $350. This gives you enforcement rights against visual lookalikes that might not literally spell "DeftBrain."

Wait for the word mark to register first — if the word mark sails through, the design mark usually follows easily on the same underlying basis.
