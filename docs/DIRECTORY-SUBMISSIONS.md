# Directory Submission Kit + Checklist

Goal: referring domains + trial users, no social media required. Two afternoons of
form-filling. As of July 2026 the site has **1** referring domain — each listing below
is one more, and several of these directories are also scraped by LLMs as "truth
sources," which helps AI-search visibility too.

**Ground rules**
- Free listings only at this stage — skip/defer anything that demands payment
  ("expedited review" upsells included). Pricing policies drift; verify on the page.
- Vary the description slightly between sites (use the variants below) — identical
  text everywhere looks scraped.
- Same logo + name everywhere for brand consistency.
- Log every submission in the tracking table at the bottom; check GSC ▸ Links
  2–4 weeks after each batch.
