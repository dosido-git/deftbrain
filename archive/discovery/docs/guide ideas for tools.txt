Key findings
The suite divides cleanly into two kinds of tools. The front-door tools (HIGH/MEDIUM fit) solve real problems people Google in real moments. The depth tools (NONE) solve problems too abstract, too playful, or too self-reflective to be Google-acquired. Both kinds are valuable — but only the first kind drives traffic through SEO guides. The NONE tier isn’t a failure; it’s the retention and depth of the suite, accessible via ToolFinder and internal navigation once a user arrives.

Three triage issues require resolution before writing overlapping guides:

SubscriptionGuiltTrip likely owns UpsellShield’s Subscriptions cluster. Eight queries in the existing backlog probably belong to a tool we hadn’t discovered yet. Reassignment is straightforward when SubscriptionGuiltTrip’s discovery pass runs.
ConflictCoach vs. DifficultTalkCoach. The 25-query estimate for ConflictCoach is only meaningful if it specializes (e.g., in response-after-the-fact de-escalation, distinct from DTC’s initiate-the-conversation scope). Without a clean split, the two tools will produce near-identical guides and cannibalize each other’s rankings.
Four additional pair triages affect smaller query shares: JargonAssassin / PlainTalk, ContrastReport / PlotTwist / DecisionCoach, HecklerPrep / TheRunthrough, ProcedureProbe / DoctorVisitTranslator, and RulebookBreaker / ComplaintEscalationWriter / UpsellShield. Each needs a clean scope boundary before multiple guides target the same query.
Several tools are deceptively low-priority despite seeming promising:

Productivity tools (BatchFlow, BrainDumpBuddy, TaskAvalancheBreaker, FocusPocus, FocusSoundArchitect) — all LOW because productivity SEO is among the most competitive verticals on the web. Even good guides rank poorly.
Cooking tools (MiseEnPlace, Giftology) — recipe-site SEO has been dominated for a decade by Pinterest-optimized content. Difficult to crack.
Fashion (WardrobeChaosHelper) — same dynamic.
These tools are still valuable inside the suite; they’re just not SEO-acquirable.
